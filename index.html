  • Entendendo Comunicação Cliente x Server

    voltar ao início

    A história da web remonta aos primeiros computadores, quando cientistas e pesquisadores começaram a explorar a conectividade entre máquinas. Esse avanço culminou no surgimento do World Wide Web, idealizada por Tim Berners-Lee em 1989. Berners-Lee, juntamente com Robert Cailliau e outros visionários, estabeleceu os fundamentos da web moderna, introduzindo conceitos como URLs, HTTP e HTML.

    No contexto da linguagem web, os clientes referem-se aos softwares que acessam e interagem com os recursos disponíveis na Internet. Isso inclui navegadores Web, como Chrome, Firefox e Safari, que permitem aos usuários visualizar e interagir com páginas web, bem como diversas aplicações Web, adaptadas para diferentes dispositivos móveis, que fornecem uma variedade de serviços e funcionalidades aos usuários.

    Os servidores são máquinas ou programas que fornecem serviços ou recursos para outros dispositivos. Eles desempenham um papel fundamental na infraestrutura da web, oferecendo serviços como hospedagem de sites, armazenamento de dados, processamento de solicitações de clientes e muito mais. Os servidores podem variar em termos de hardware e software, abrangendo desde servidores web simples até servidores de banco de dados robustos.

    Linguagens de programação desempenham um papel crucial na criação de conteúdo web dinâmico e interativo. Existem distinções importantes entre linguagens server-side, que são processadas no servidor, e linguagens client-side que são processadas no navegador do cliente. Embora o HTML seja essencial na estruturação de páginas da web, não é considerado uma linguagem de programação, pois ele é uma linguagem de marcação. Linguagens como JavaScript e Python são amplamente utilizadas para criar interatividade e funcionalidades dinâmicas tanto no servidor quanto no cliente, impulsionando a evolução da web moderna.

  • Introdução ao HTML na prática

    voltar ao início

    O Visual Studio Code é um editor de código-fonte desenvolvido pela Microsoft e amplamente utilizado por desenvolvedores web. Ele oferece uma ampla gama de recursos, incluindo realce de sintaxe, controle de versão integrado e suporte a extensões, tornando-o uma escolha popular no desenvolvimento web.

    A extensão Live Server do Visual Studio Code permite criar um servidor de desenvolvimento local com atualizações em tempo real. Com essa extensão, é possível visualizar as alterações feitas no código HTML, CSS ou JavaScript imediatamente no navegador sem a necessidade de atualizar a página manualmente, o que agiliza o processo de desenvolvimento.

    O Inspetor de Elementos do Navegador é uma ferramenta presente nos navegadores web que permite visualizar o código HTML, CSS e JavaScript de uma página em tempo real. Com o inspetor de elementos, os desenvolvedores podem analisar a estrutura da página, identificar problemas de layout, depurar código e fazer ajustes de estilo diretamente no navegador.

    AEstrutura Básica do HTML consiste em elementos fundamentais que definem a estrutura e o conteúdo de uma página web. Geralmente, inclui a declaração do tipo de documento (<!DOCTYPE html>), o elemento <html> que encapsula todo o conteúdo da página, o elemento <head> que contém metadados e referências a recursos externos, e o elemento <body> que contém o conteúdo visível da página.

    Tags HTML são elementos usados para estruturar e formatar o conteúdo de uma página web. Elas são representadas por meio de pares de tags, como <tag> e </tag>, onde o primeiro indica o início e o segundo indica o fim do elemento. Exemplos de tags incluem <h1> para cabeçalhos, <p> para parágrafos, <a>d para links e <img> para imagens.

    Atributos das Tags são informações adicionais fornecidas às tags HTML para modificar seu comportamento ou aparência. Eles são especificados dentro do elemento de abertura da tag e consistem em um nome seguido por um valor, separados por um sinal de igual (=). Exemplos de atributos incluem href para links, src para imagens e class para aplicar estilos CSS.

    Para inserir Textos em uma página HTML, pode-se utilizar o elemento <p> para parágrafos, <h1> a <h6> para cabeçalhos de diferentes níveis, e <span> para agrupar partes de texto para aplicar estilos específicos. Além disso, é possível utilizar elementos de formatação de texto, como <strong> para texto em negrito e <em> para texto em itálico.

    HTML oferece suporte a dois tipos principais de listas: listas ordenadas (<ol>) e listas não ordenadas (<ul>). Em uma lista ordenada, cada item é numerado sequencialmente, enquanto em uma lista não ordenada, os itens são marcados com pontos ou outro símbolo. Os itens de uma lista são definidos pelo elemento >li<.

    Os links em HTML são criados usando o elemento <a>', que possui o atributo href para especificar o destino do link. Os links podem apontar para páginas web externas, arquivos locais, seções específicas de uma página (ancoragem) ou endereços de e-mail. É comum incluir também o atributo target="_blank" para abrir o link em uma nova aba ou janela do navegador.
